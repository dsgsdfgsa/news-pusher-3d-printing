# 3D 打印新闻推送系统 - 完整设计方案

## 📋 项目概述

**需求描述：**
- 每天早上 8:30 自动推送昨日 3D 打印领域热点新闻
- 通过邮件发送到指定邮箱
- 新闻内容需筛选、整理和汇总

**目标：**
- 建立自动化新闻抓取和推送系统
- 确保新闻质量和时效性
- 提供稳定可靠的推送服务

---

## 🎯 核心功能

1. **新闻抓取**：自动从多个来源抓取 3D 打印相关新闻
2. **新闻筛选**：智能筛选热点和高价值新闻
3. **内容整理**：自动生成新闻摘要和分类
4. **定时推送**：每天 8:30 准时发送邮件
5. **历史记录**：保存推送历史，便于回溯

---

## 🛠️ 技术方案选择

### 方案对比

| 方案 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| **方案 1：本地 Python 脚本** | 免费、可控性强、灵活 | 需要持续运行电脑、邮件发送需要 SMTP 配置 | 个人使用、电脑长期在线 |
| **方案 2：云函数 + 定时器** | 无需服务器、按量付费、稳定可靠 | 需要云服务账号、有基础费用 | 生产环境、稳定性要求高 |
| **方案 3：GitHub Actions** | 免费、自动化、版本控制 | 执行时间限制、需要 GitHub 账号 | 开源项目、轻量级任务 |
| **方案 4：RSS 订阅 + 过滤器** | 原生支持、无需开发 | 灵活性低、定制化困难 | 简单需求、无需复杂处理 |

---

## ✅ 推荐方案：云函数 + 定时器

**选择理由：**
- ✅ 无需本地电脑持续运行
- ✅ 成本低（按量付费，通常免费额度足够）
- ✅ 稳定可靠，不受本地环境影响
- ✅ 易于维护和扩展

**云服务推荐：**
1. **阿里云函数计算 FC** + **定时触发器**（推荐国内）
2. **腾讯云函数 SCF** + **定时触发器**（推荐国内）
3. **AWS Lambda** + **EventBridge**（国外）
4. **Google Cloud Functions** + **Cloud Scheduler**（国外）

# index.py
import json
import requests
from datetime import datetime, timedelta
from email.mime.text import MIMEText
from email.mime.multipart import MIMEMultipart
import smtplib
import os
import re
from bs4 import BeautifulSoup

# 环境变量配置
SMTP_HOST = os.environ.get('SMTP_HOST')
SMTP_PORT = int(os.environ.get('SMTP_PORT', 465))
SMTP_USER = os.environ.get('SMTP_USER')
SMTP_PASSWORD = os.environ.get('SMTP_PASSWORD')
TO_EMAIL = os.environ.get('TO_EMAIL')

# 新闻源配置
NEWS_SOURCES = {
    '3D打印行业网': {
        'url': 'https://www.3dprintingindustry.com/news/',
        'selector': '.post-title a'
    },
    'TCT亚洲展': {
        'url': 'https://www.tctasia.com/news/',
        'selector': '.news-item a'
    },
    '南极熊3D打印': {
        'url': 'https://www.nanjixiong.com/',
        'selector': '.article-list a'
    }
}

# 搜索关键词（用于筛选3D打印相关新闻）
KEYWORDS = [
    '3D打印', '3d printing', '增材制造', 'additive manufacturing',
    'FDM', 'SLA', 'SLM', 'DLP', '金属打印', '树脂打印',
    '打印机', '打印材料', '切片软件'
]

def fetch_news():
    """抓取新闻"""
    news_items = []
    yesterday = (datetime.now() - timedelta(days=1)).strftime('%Y-%m-%d')

    for source_name, config in NEWS_SOURCES.items():
        try:
            print(f"正在抓取 {source_name}...")
            response = requests.get(config['url'], timeout=10)
            response.encoding = 'utf-8'

            soup = BeautifulSoup(response.text, 'html.parser')
            articles = soup.select(config['selector'])

            for article in articles[:10]:  # 每个源最多取10条
                title = article.get_text(strip=True)
                link = article.get('href', '')

                if not link.startswith('http'):
                    link = config['url'].rstrip('/') + '/' + link.lstrip('/')

                # 筛选相关新闻
                if any(keyword.lower() in title.lower() for keyword in KEYWORDS):
                    news_items.append({
                        'source': source_name,
                        'title': title,
                        'link': link,
                        'date': yesterday
                    })

        except Exception as e:
            print(f"抓取 {source_name} 失败: {e}")

    return news_items

def summarize_news(news_items):
    """生成新闻摘要"""
    if not news_items:
        return "暂无相关新闻。"

    # 按来源分组
    grouped = {}
    for item in news_items:
        source = item['source']
        if source not in grouped:
            grouped[source] = []
        grouped[source].append(item)

    # 生成 HTML 内容
    html = """
    <html>
    <head>
        <style>
            body { font-family: 'Microsoft YaHei', Arial, sans-serif; line-height: 1.6; color: #333; }
            .container { max-width: 800px; margin: 0 auto; padding: 20px; }
            .header { background: linear-gradient(135deg, #667eea 0%, #764ba2 100%); color: white; padding: 30px; border-radius: 10px; margin-bottom: 30px; }
            .header h1 { margin: 0; font-size: 28px; }
            .header .date { margin-top: 10px; opacity: 0.9; }
            .section { margin-bottom: 30px; }
            .section h2 { color: #667eea; border-bottom: 2px solid #667eea; padding-bottom: 10px; }
            .news-item { padding: 15px; margin: 10px 0; background: #f8f9fa; border-left: 4px solid #667eea; border-radius: 5px; }
            .news-item a { color: #667eea; text-decoration: none; font-weight: bold; }
            .news-item a:hover { text-decoration: underline; }
            .source { color: #999; font-size: 14px; margin-top: 5px; }
            .footer { text-align: center; color: #999; padding: 20px; border-top: 1px solid #eee; margin-top: 30px; }
        </style>
    </head>
    <body>
        <div class="container">
            <div class="header">
                <h1>📰 3D 打印热点新闻</h1>
                <div class="date">{date}</div>
            </div>
    """.format(date=datetime.now().strftime('%Y年%m月%d日'))

    for source, items in grouped.items():
        html += f"""
            <div class="section">
                <h2>{source}</h2>
        """
        for item in items[:5]:  # 每个来源最多5条
            html += f"""
                <div class="news-item">
                    <a href="{item['link']}" target="_blank">{item['title']}</a>
                    <div class="source">{item['date']}</div>
                </div>
            """
        html += "</div>"

    html += """
            <div class="footer">
                <p>本邮件由自动推送系统发送，如有问题请联系管理员。</p>
                <p>🚀 Powered by AI News Pusher</p>
            </div>
        </div>
    </body>
    </html>
    """

    return html

def send_email(subject, content):
    """发送邮件"""
    msg = MIMEMultipart('alternative')
    msg['Subject'] = subject
    msg['From'] = SMTP_USER
    msg['To'] = TO_EMAIL

    # 添加 HTML 内容
    html_part = MIMEText(content, 'html', 'utf-8')
    msg.attach(html_part)

    # 发送邮件
    try:
        if SMTP_PORT == 465:
            # SSL
            server = smtplib.SMTP_SSL(SMTP_HOST, SMTP_PORT)
        else:
            # TLS
            server = smtplib.SMTP(SMTP_HOST, SMTP_PORT)
            server.starttls()

        server.login(SMTP_USER, SMTP_PASSWORD)
        server.send_message(msg)
        server.quit()
        print("邮件发送成功！")
        return True
    except Exception as e:
        print(f"邮件发送失败: {e}")
        return False

def handler(event, context):
    """函数入口"""
    print("开始执行 3D 打印新闻推送任务...")

    # 1. 抓取新闻
    print("步骤 1: 抓取新闻...")
    news_items = fetch_news()
    print(f"共抓取到 {len(news_items)} 条新闻")

    # 2. 生成摘要
    print("步骤 2: 生成新闻摘要...")
    content = summarize_news(news_items)

    # 3. 发送邮件
    print("步骤 3: 发送邮件...")
    yesterday = (datetime.now() - timedelta(days=1)).strftime('%Y-%m-%d')
    subject = f"📰 3D打印热点新闻 - {yesterday}"
    success = send_email(subject, content)

    if success:
        return {
            'statusCode': 200,
            'body': json.dumps({
                'message': '推送成功',
                'news_count': len(news_items),
                'email': TO_EMAIL
            })
        }
    else:
        return {
            'statusCode': 500,
            'body': json.dumps({
                'message': '推送失败',
                'error': '邮件发送失败'
            })
        }
```

def categorize_news(news_items):
    """分类新闻"""
    categories = {
        '技术突破': ['技术', '突破', '创新', '研发'],
        '市场动态': ['市场', '融资', '合作', '收购'],
        '产品发布': ['发布', '新品', '推出', '上市'],
        '行业资讯': ['展会', '活动', '政策', '趋势']
    }

    categorized = {cat: [] for cat in categories.keys()}

    for item in news_items:
        title = item['title']
        for category, keywords in categories.items():
            if any(kw in title for kw in keywords):
                categorized[category].append(item)
                break
        else:
            categorized['行业资讯'].append(item)

    return categorized
```

import sqlite3

def init_database():
    """初始化数据库"""
    conn = sqlite3.connect('/tmp/news_history.db')
    cursor = conn.cursor()

    cursor.execute('''
        CREATE TABLE IF NOT EXISTS news_history (
            id INTEGER PRIMARY KEY AUTOINCREMENT,
            title TEXT,
            link TEXT,
            source TEXT,
            date TEXT,
            sent_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
        )
    ''')

    conn.commit()
    conn.close()

def save_news(news_items):
    """保存新闻历史"""
    conn = sqlite3.connect('/tmp/news_history.db')
    cursor = conn.cursor()

    for item in news_items:
        cursor.execute('''
            INSERT INTO news_history (title, link, source, date)
            VALUES (?, ?, ?, ?)
        ''', (item['title'], item['link'], item['source'], item['date']))

    conn.commit()
    conn.close()

## ⚠️ 注意事项

### 1. 邮件发送限制

- **QQ 邮箱**：每天 500 封
- **163 邮箱**：每天 200 封
- **Gmail**：每天 500 封
- 如果超出限制，考虑使用企业邮箱

### 2. 新闻源可用性

- 网站可能更改结构，导致爬虫失效
- 建议定期检查和更新新闻源
- 使用多个源避免单点故障

### 3. 时区问题

- 定时任务使用 UTC 时间
- 北京时间 = UTC + 8
- 例如：北京时间 8:30 = UTC 0:30

### 4. 反爬虫

- 部分网站有反爬虫机制
- 添加 User-Agent 和请求头
- 控制请求频率，避免被封

headers = {
    'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36'
}
response = requests.get(url, headers=headers, timeout=10)
